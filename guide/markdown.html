You need to build your document or blog page by creating and writing Markdown in the corresponding path.

# Theme Enhancement

# Enable all

You can set plugins.mdEnhance.enableAll: true in theme options to enable all features of the md-enhanceopen in new window plugin.

const { hopeTheme } = require("vuepress-theme-hope");

module.exports = {
  theme: hopeTheme({
    plugins: {
      mdEnhance: {
        enableAll: true,
      },
    },
  }),
};
