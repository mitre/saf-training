# InSpec Beginner

<CourseVersion />


> Learn InSpec fundamentals and start writing security tests

## What You'll Learn

By the end of this course, you'll be able to:

- ✅ Understand InSpec profile architecture
- ✅ Write InSpec controls for security testing
- ✅ Use inputs to parameterize tests
- ✅ Generate and analyze InSpec results
- ✅ Create overlays from existing profiles
- ✅ Convert STIGs to InSpec profiles

## Prerequisites

::: warning Before You Start
Basic command line experience
Understanding of security concepts
Familiarity with Ruby syntax (helpful but not required)
:::
