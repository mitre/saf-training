# InSpec Advanced

> Build custom InSpec resources and integrate into CI/CD pipelines

## What You'll Learn

By the end of this course, you'll be able to:

- ✅ Create custom InSpec resources
- ✅ Extend InSpec to test new components
- ✅ Integrate InSpec into CI/CD pipelines
- ✅ Use GitHub Actions for automated testing
- ✅ Validate results with SAF CLI
- ✅ Contribute resources back to InSpec

## Prerequisites

::: warning Before You Start
Completion of InSpec Beginner course
Ruby programming experience
Git and GitHub knowledge
CI/CD pipeline familiarity
:::
