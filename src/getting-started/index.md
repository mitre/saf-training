# SAF User Class

<CourseVersion />


> Learn to automate security validation and visualization with the MITRE Security Automation Framework

## What You'll Learn

By the end of this course, you'll be able to:

- ✅ Identify and locate security guidance for software components
- ✅ Understand the SAF pillars: Plan, Harden, Validate, Normalize, Visualize
- ✅ Run InSpec profiles to validate security compliance
- ✅ Visualize security results in Heimdall
- ✅ Export compliance data for reporting
- ✅ Integrate SAF tools into your security workflow

## Who This Course Is For

This course is designed for:
- **Security professionals** looking to automate compliance testing
- **System administrators** managing secure infrastructure
- **Compliance officers** needing automated validation
- **Anyone** new to the Security Automation Framework

No coding experience required!
