# STIG Development

> Create STIG-ready security guidance with Vulcan

## What You'll Learn

By the end of this course, you'll be able to:

- ✅ Understand STIG structure and components
- ✅ Use Vulcan to create security guidance
- ✅ Write checks and fixes for requirements
- ✅ Combine and organize requirements
- ✅ Generate automated InSpec tests
- ✅ Export and publish STIGs

## Prerequisites

::: warning Before You Start
Understanding of security requirements
Familiarity with STIGs and SRGs
Basic YAML and markdown knowledge
:::
